# 環境構築手順

## 必要なソフトウェア

- Node.js LTS (v24.12 で動作確認)
- npm (Node.js に同梱)
- Git
- Visual Studio Code (推奨)

## セットアップ手順(Windows)

1. Node.js と npm インストール

   1. [Node.js 公式サイト](https://nodejs.org/ja)からダウンロード
   1. インストール実行
   1. PowerShell を開き以下を実行

      ```bash
      node --version
      v24.12.x

      npm --version
      11.x.x
      ```

1. プロジェクトセットアップ

   ```bash
   git clone https://github.com/masakiYLv1/errordict.git
   cd errordict
   npm install
   ```

1. 環境変数の設定
   プロジェクト直下に `.env.local` ファイルを作成し、以下を設定します

   ```env
   VITE_SUPABASE_URL=your_supabase_project_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

1. 開発サーバー起動

   ```bash
   npm run dev
   ```

1. 動作確認

   - [http://localhost:5173] でアクセス確認
   - エラー一覧ページが表示されることを確認
