# 画面設計書

## 1. 画面一覧

### フロントエンド

| 画面名           | URL              | 説明           |
| ---------------- | ---------------- | -------------- |
| エラー一覧ページ | /errors          | エラー一覧表示 |
| エラー詳細ページ | /errors/:id      | エラー詳細表示 |
| エラー新規登録   | /errors/new      | エラー新規登録 |
| エラー編集       | /errors/:id/edit | エラー編集     |

### 認証(Phase2)

| 画面名           | URL            | 説明             |
| ---------------- | -------------- | ---------------- |
| ログイン         | /auth/login    | ユーザー認証     |
| ユーザー新規登録 | /auth/register | ユーザー新規登録 |

## 2. レスポンシブ対応

- 対象デバイス：PC / スマートフォン
  - モバイル：～ 767px
  - PC：768px ～
- 対応内容：
  - margin 調整
  - padding 調整
  - fontsize 調整

### エラー編集ページ /errors/:id/edit

- 内容は新規登録ページと同じ
- 既存データが入力済みの状態
- [← 一覧へ戻る]の箇所が[←]となる
- [登録]の箇所が[更新]となる
